
# Food Delivery Data Analysis
## Overview
This project contains an analysis of a food delivery dataset using Python and Pandas. The main aim is to understand customer ordering behavior, restaurant performance, and revenue trends by working with real-world style data.

The analysis answers practical questions related to orders, users, memberships, and restaurants, similar to those asked in data analytics assignments and lab examinations.

---

## Data Used

The project uses data from multiple sources:

* Orders data in CSV format
* User details in JSON format
* Restaurant information stored in a SQL table

These datasets are merged using common keys to create a single dataset for analysis.

---

## Tools

* Python
* Pandas
* SQL (SQLite)
* Jupyter Notebook

## What This Project Covers

* Order and revenue analysis by city
* Comparison between Gold and Regular users
* Cuisine and restaurant performance analysis
* Handling joins and missing values

## How to Run

Open the notebook and run the cells in order to reproduce the results.

All results are generated using logical data operations without hard-coding values. The code is written in a simple and readable way for easy understanding.
